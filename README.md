## Design principals

1. Make it for all
  * Build it, test it and improve it — over and over again
  * Use plain english
  * Use HTML to guide screenreaders though the content easily
  * Use Aria
  * Make sure colours contrast enough
  * Use readable text sizes
2. Enhance, don't require
  * Content should load first
  * The site should be useable without CSS or Javascript
  * Fancy effects are fine, as long as they're not detrimental to performance
3. Web-first
  * Design in the browser
  * Develop in the open
